BakeSmart is a full-stack online bakery developed using Python, Django, MongoDB, and React.js.
The application allows users to browse bakery products, manage carts, place orders, generate bills and receive personalized product recommendations based on order history.If user login as Admin it allows user to do CRUD operations on products.
This project was developed as an Individual Project during Semester 4, focusing on full-stack development, backend logic and database integration.
